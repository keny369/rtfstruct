If you believe you have found a security issue in `rtfstruct`, do not open a public issue containing exploit details.

Contact Lee Powell through Lumen & Lever:

https://lumenandlever.com/contact

Please include:

- affected version
- input type
- reproduction steps
- expected behavior
- observed behavior
- any relevant fixture or minimized input where safe to share
